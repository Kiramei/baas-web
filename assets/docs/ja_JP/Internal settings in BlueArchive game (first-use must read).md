
# ⚙️ Blanche Files — ゲーム内設定ガイド（初回必読）

本ガイドは、**BAAS自動化システム**と連携させるために必要な
**必須設定**および**推奨設定**を示します。

---

## 🧩 必須設定

| カテゴリ           | 設定項目           | 値／指示                    |
| :------------- | :------------- | :---------------------- |
| **グラフィック**     | 縦画面バトルテキストボックス | **オフ**                  |
| **回想ロビー**      | キャラ選択          | **Ako・Aru・Wakamoを選ばない** |
| **言語（グローバル版）** | —              | **English**             |

---

## 💡 推奨設定（任意）

| 設定           | 推奨値    |
| :----------- | :----- |
| **解像度**      | 最高     |
| **FPS**      | 60     |
| **描画モード**    | 互換性モード |
| **ポストプロセス**  | ON     |
| **アンチエイリアス** | ON     |

---

✅ **ヒント:** 高画質設定は認識精度を安定させますが、動作安定性には必須ではありません。
