
# 🕒 调度配置填写指南

---

## ⚙️ 1. 优先级

队列中存在多个任务时，**优先级数值越小，越先执行**。

---

## ⏳ 2. 执行间隔

* 整数 `0` 表示每日执行一次；
* 更大的整数代表更长天数间隔。

---

## 🕐 3. 每日定时执行

任务每日固定时间执行（可多时段，以逗号分隔）。

格式：

```
[[h, m, s]]  （UTC 时间）
```

示例：

```
[[0, 0, 0], [20, 0, 0]]
```

表示北京时间（UTC+8）上午 8 点与下午 4 点执行。

---

## 🚫 4. 禁止执行时间段

任务在特定时间段内不会运行。
格式：

```
[[[h1, m1, s1], [h2, m2, s2]]]
```

示例：

```
[[[0,0,0],[24,0,0]]]
```

表示全天禁用。

---

## 🔁 5. 前置与后置任务

* **前置任务：** 在当前任务前执行；
* **后置


任务：** 在当前任务后执行。

保证依赖任务顺序正确。
